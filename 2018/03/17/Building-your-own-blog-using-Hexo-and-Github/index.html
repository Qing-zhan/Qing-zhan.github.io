Take-away

经过一些设置后，以后每次发博文的流程如下:

  1. 打开 CMD 并进入 blog 目录，执行 hexo new post "post-name"
  2. 进入 source/posts_/ 打开对应的 md 文件，文件头中添加 tags 标签和 categories 分类，并编辑内容
  3. 在 CMD 中执行hexo d -g 完成部署和生成
  4. 进入网页查看或者使用 hexo s 在 localhost:4000 中查看
