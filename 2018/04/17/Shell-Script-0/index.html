4. Double quote & Single quote

Notation $ is a build-in notation, when you want to use the $ alone, escape character \ can be used.

Quoting mechanism allows otherwise special characters, such as whitespace, wildcards, parentheses, and dollar signs to be taken as literal text.
In double quote "" (weak quote), $name is expanded as a variable. In weak quote, enclosed whitespace and wildcards are taken as literals, but variables and command substitutions are still performed; While in single quote '' (strong quote), nothing is expanded, the entire string is taken as literal. This is the main difference between them.
