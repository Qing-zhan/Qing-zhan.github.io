Git Tutorial

Posted on 2018-04-14 | In Software

1. SSH key generation

在Git中，本地版本库与远程的Github版本库的连接通过SSH加密协议。主要的步骤就是在本地生成SSH Key，然后添加到Github的可信任列表中，此步骤在网上可以很容易找到，故不再赘述。
